# Highway Cops Base Game (Scroll down for update) - Mahdiat Khan

# INTRODUCTION

This project involves the design and documentation of a procedural highway driving game aimed to be created within Unity 3D. The player will be driving along an endless highway while earning points, points put into a currency system to then buy upgrades or better cars.

# GAME OVERVIEW

The game focuses on an endless highway environment where the player must drive for as long as possible to earn more points. The original scope of this project was to make a police pursuit based game in a city environment, but it was far too out of scope to try and learn the new material and resources needed to build a game with that vision, so there was a shift in focus to this highway based game instead, currently lacking any sort of police/pursuit AI, but being a more realistic approach it was chosen as a suitable replacement.
